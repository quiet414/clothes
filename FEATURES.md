# 🎯 功能演示指南

## 🚀 快速开始

1. **打开应用** - 双击 `index.html` 在浏览器中启动
2. **浏览功能** - 使用顶部导航切换不同功能模块
3. **开始体验** - 从虚拟试衣间开始您的穿搭之旅

## 🎨 3D虚拟人物定制

### 外观定制
1. 点击虚拟试衣间的 "🎨 定制形象" 按钮
2. **肤色选择** - 点击不同肤色选项
3. **发型选择** - 选择短发、长发或马尾
4. **发色选择** - 从6种发色中选择
5. **旋转控制** - 使用按钮或拖拽旋转虚拟人物

### 身材调整
1. 点击 "📏 调整身材" 按钮
2. 使用滑块调整各项身体参数
3. 实时查看3D人物的身材变化
4. 点击 "保存设置" 应用更改

## 👗 衣橱管理

### 添加新衣服
1. 点击 "📷 拍照上传" 或 "📁 导入图片"
2. 选择或拍摄衣服照片
3. 设置衣服类型、名称和颜色
4. 点击 "保存衣服" 添加到衣橱

### 分类筛选
1. 使用分类标签快速筛选衣服
2. **全部** - 显示所有衣服
3. **上衣/下装/连衣裙/外套/鞋子/配饰** - 分类查看

## ✨ 虚拟试衣

### 试穿衣服
1. 点击衣橱中的任意衣服
2. 观察虚拟人物身上的穿搭效果
3. 可以叠加多件衣服进行搭配

### 搭配管理
- **🎲 随机搭配** - AI智能推荐随机搭配
- **💾 保存搭配** - 保存当前满意的搭配
- **🔄 重置** - 清除当前所有搭配

## 💝 搭配方案

### 查看方案
1. 点击顶部 "💝 搭配方案" 导航
2. 查看所有保存的搭配方案
3. 查看每个方案的详细信息

### 管理方案
- **穿上** - 一键应用保存的搭配
- **删除** - 移除不需要的搭配方案

## 🌟 达人穿搭推荐

### 浏览推荐
1. 点击顶部 "🌟 穿搭推荐" 导航
2. 浏览各种风格的穿搭案例

### 风格筛选
- **全部风格** - 显示所有推荐
- **休闲风** - 日常休闲穿搭
- **优雅风** - 正式优雅搭配
- **街头风** - 时尚潮流风格
- **职场风** - 商务职业搭配

### 使用推荐
- **试穿** - 模拟试穿达人搭配
- **收藏** - 保存喜欢的搭配到个人方案
