# 🛍️ 3D虚拟试衣间 - 模拟人生风格的衣橱管理应用

这是一个具有《模拟人生》游戏风格的3D虚拟试衣间应用，支持真实的身材比例调整、3D服装试穿和淘宝风格的商城购物体验。

## 🌟 主要功能

### 🎯 3D Avatar系统
- **模拟人生风格**：类似《模拟人生》游戏的3D人物外观和材质效果
- **真实身材比例**：基于身高、体重、肩宽、胸围、腰围、臀围等6项参数动态调整3D模型
- **360度交互**：支持鼠标拖拽旋转和滚轮缩放查看
- **实时渲染**：高质量的光影效果和材质渲染

### 👕 真实服装系统
- **3D服装模型**：包含T恤、衬衫、牛仔裤、连衣裙、外套、鞋子等多种真实服装
- **材质渲染**：支持棉质、牛仔、丝绸、皮革等不同材质效果
- **颜色定制**：多种颜色选择，支持自定义配色
- **纹理贴图**：支持真实商品图片作为服装纹理
- **智能贴合**：服装自动贴合3D人体模型

### 🛒 淘宝风格商城
- **商品分类**：上衣、下装、连衣裙、外套、鞋子、配饰等6大分类
- **真实商品**：包含真实价格、品牌、评分、销量等信息
- **商品详情**：详细的商品描述、颜色选择、尺码选择
- **一键添加**：直接添加商品到个人衣橱
- **购买链接**：集成真实的购买功能

### 💾 智能存储
- **本地存储**：所有数据保存在浏览器本地存储中
- **搭配方案**：支持保存和加载多种搭配方案
- **数据持久化**：关闭浏览器后数据不丢失

## 🚀 快速开始

### 本地运行
1. 将项目文件放在Web服务器目录下（需要HTTP服务器）
2. 在浏览器中打开 `index.html`
3. 等待3D引擎加载完成
4. 开始使用应用

### 在线体验
将项目部署到支持静态网站托管的平台：
- GitHub Pages
- Netlify
- Vercel
- 阿里云OSS

### 测试页面
- 打开 `test.html` 进行功能测试和系统检测

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计，包含响应式布局
- **JavaScript** - 交互功能
- **Three.js** - 3D渲染引擎
- **OrbitControls** - 3D交互控制
- **LocalStorage** - 本地数据存储
- **File API** - 图片上传和处理

## 📱 功能演示

### 1. 3D Avatar设置
- 点击"调整身材"按钮
- 使用滑块调整各项身体参数
- 实时查看3D人物的身材变化
- 点击"保存设置"应用更改

### 2. 3D服装试穿
- 点击衣橱中的衣服进行试穿
- 3D人物实时显示穿着效果
- 支持多层服装叠加显示
- 使用"随机搭配"获得搭配建议

### 3. 淘宝商城购物
- 点击"浏览商城"进入淘宝风格界面
- 浏览不同分类的真实商品
- 查看商品详情、价格、评价
- 一键添加到个人衣橱

### 4. 保存搭配方案
- 调整好满意的搭配后
- 点击"保存搭配"
- 为搭配方案命名
- 在"搭配方案"页面查看和管理

## 🎨 界面特色

- **模拟人生风格** - 类似游戏的3D视觉效果
- **现代化设计** - 简洁美观的UI界面

## 💾 数据存储

应用使用浏览器的LocalStorage存储数据：
- 用户身材参数
- 衣服图片和信息
- 保存的搭配方案
- 3D模型配置数据

数据仅在本地保存，保护用户隐私。

## 🔧 自定义开发

### 添加新功能
1. 在 `script.js` 中添加功能函数
2. 在 `3d-avatar.js` 中扩展3D功能
3. 在 `index.html` 中添加对应的UI元素
4. 在 `style.css` 中添加样式

### 添加新服装
1. 在 `clothing-models.js` 中创建新的3D服装模型
2. 定义材质、几何体和纹理
3. 在商城系统中添加商品信息

### 修改3D效果
- 编辑 `3d-avatar.js` 文件
- 调整光照、材质和渲染参数
- 优化3D模型的比例和细节

### 集成后端
- 替换LocalStorage为API调用
- 添加用户认证功能
- 支持云端数据同步
- 集成真实的商品API
